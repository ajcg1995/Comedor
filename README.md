# Comedor
Sistema de control para comedores estudiantiles.
Sistema Diseñado en C# cuenta con las capas BLL, DAL, UI.
Base de datos SQL SERVER

Descripcion: Sistema creado para llevae el control de lo estudiantes que hacen uso del comedor estudiantil.
Cuenta con la administracion de Alumnnos, y guarda un historial con la fecha y los datos del estudiantes, ademas permite la
impresion de un tickete en una impresora POS
